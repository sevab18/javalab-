# javalab-

## Lab 1: Основы ООП

- Созданы базовые классы `Product` и `Category`.
- Реализована основная логика для управления товарами и категориями.

## Lab 2: Инкапсуляция и Валидация

- **Инкапсуляция:** Все поля в классе `Product` сделаны приватными (`private`).
- **Геттеры:** Для всех полей добавлены публичные геттеры.
- **Защищенные сеттеры (Guarded Mutators):**
  - Вместо обычных сеттеров реализованы методы `trySet...()`, которые возвращают `boolean` (`true` при успехе, `false` при ошибке).
  - Добавлена валидация для всех полей `Product` (ID, имя, цена, количество и т.д.).
- **Обновлены методы инвентаризации:**
  - `addStock()`, `sellProduct()`, `applyDiscount()` теперь используют защищенные сеттеры и возвращают `boolean`.
- **Вычисляемый статус запасов:**
  - `getStockStatus()` теперь является вычисляемым полем на основе количества товара.
- **Безопасность в `Category`:**
  - Метод `addProduct()` в классе `Category` теперь возвращает `boolean` и предотвращает добавление `null` или дубликатов.
- **Демонстрация:** Файл `ShopDemo.java` обновлен для демонстрации как успешных, так и неудачных операций, подтверждая работу валидации.
